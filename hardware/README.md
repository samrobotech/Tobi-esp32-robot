# Hardware

This directory contains the hardware resources for the **Tobi ESP32 Desktop Robot**.

## Contents

- Electronic schematics
- Wiring diagrams
- PCB design files (if available)
- Hardware documentation

## Hardware Overview

The robot is built around an ESP32 microcontroller and includes:

- ESP32 development board
- OLED display
- Capacitive touch sensor
- Power management circuit

For assembly instructions and firmware, please refer to the main project documentation.
