# 🤖 Tobi ESP32 Desktop Robot

*A compact desktop companion robot powered by ESP32, featuring expressive OLED animations and capacitive touch interaction.*

# 📖 Overview

Tobi is a compact desktop companion robot designed around the ESP32 microcontroller. It combines expressive OLED eye animations with capacitive touch interaction to create a simple yet engaging robotic companion.

Touching the robot's head instantly changes its behavior, allowing it to switch between multiple emotional and animation modes.

The project is fully open-source and includes firmware, electronic schematics, and 3D printable enclosure files.

# 🛠 Hardware

| Component | Description |
|-----------|-------------|
| MCU | ESP32 | like :s3 , c3 mini , ...
| Display | SSD1306 OLED |0.96 oled
| Touch Sensor | tb sensor touch (capasitor sensor) |

# 🚀 Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/samrobotech/tobi-esp32-robot.git
```

## 2. Open Firmware

Open the Arduino project located in:

```
firmware/
```

## 3. Install Libraries

- Adafruit GFX
- Adafruit SSD1306
- Wire

## 4. Upload

Select your ESP32 board and upload the firmware.
